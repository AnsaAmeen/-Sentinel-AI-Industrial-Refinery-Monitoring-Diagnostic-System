# -Sentinel-AI-Industrial-Refinery-Monitoring-Diagnostic-System
Autonomous Industrial IoT system built with n8n, AWS, and Gemini AI. Monitors refinery telemetry (Temp, Pressure, Vibration) in real-time, performs AI-driven root-cause diagnostics, and automates multi-channel alerting via Slack and Email. Includes fail-safe error handling and automated data logging for predictive maintenance.
